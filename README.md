A Machine Learning based Spam Message Detection system using **Naive Bayes** and **Flask**.

## Features
- Detects Spam or Safe messages
- Simple & clean UI
- Trained on SMS Spam dataset

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML, CSS

## How to Run
1. Clone the repository
2. Install dependencies  
   `pip install -r requirements.txt`
3. Run the app  
   `python app.py`
4. Open browser:  
   `http://127.0.0.1:5000`

## Output
- Shows whether message is **SPAM** or **SAFE**
